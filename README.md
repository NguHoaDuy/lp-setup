# Hướng dẫn cài đặt tool

## 1. Mở PowerShell: 

```html
Start -> PowerShell
```
## 2. Gõ lệnh: 

```powershell
Set-ExecutionPolicy RemoteSigned -scope CurrentUser
```

## 3. Đi tới file set_env.ps1

```powershell
cd <đường-dẫn-tới-file-ps1>
```

## 4. Chạy file set_env.ps1

```powershell
.\set_env.ps1
```

### Note:
Chọn "Yes" hoặc "Y" nếu lúc chạy file có yêu cầu.
